# Wildlife-Tracker
## Description
Implementing a wild life tracker to check animals in danger based on age and specie: it is an object/function based project
## Features
- add, edit and remove species/age/habitat etc..
- search for habitat/age/endanger
- store data if and when needed
- easy-to-use interface
## Technologies Used
- Fully built in JavaScript
